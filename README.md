# Chess Academy Website

A modern Vue.js-based website for Chess Academy in Chennai, India. The site showcases chess coaching services including group classes and one-to-one sessions.

## Technology Stack

- **Framework**: Vue 3 with Composition API
- **Build Tool**: Vite 5
- **Testing**: Vitest with Vue Test Utils
- **Styling**: CSS with CSS Variables (Chess Green Theme)

## Project Structure

```
src/
  ├── assets/
  │   └── styles.css       # Global styles
  ├── components/
  │   ├── HeaderNav.vue    # Navigation header
  │   ├── HeroSection.vue  # Hero banner
  │   ├── AboutSection.vue # About us section
  │   ├── CoursesSection.vue # Courses display
  │   ├── ContactSection.vue # Contact information
  │   └── FooterSection.vue  # Page footer
  │   └── __tests__/       # Component tests
  ├── App.vue              # Root component
  └── main.js              # Application entry point
```

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm run test

# Build for production
npm run build

# Preview production build
npm run preview
```
